# User Game Preferences

## Project: Prince of Kiro Platformer

### Technical Preferences
- **Language**: Vanilla JavaScript with HTML5 Canvas
- **File Structure**: Separate files (HTML, CSS, JS)

### Game Design Choices
- **Scoring System**: Time-based (faster completion = better score)
- **Character Type**: Pure platforming (run, jump, avoid traps - no combat)
- **Level Count**: 2 levels to start
- **Goal**: Find the Princess of Kiro (wearing Kiro purple #790ECB)

### Lives & Death System
- **Starting Lives**: 3
- **Death Behavior**: Restart at beginning of current level
- **Extra Lives**: Hidden in difficult-to-reach areas

### Collectibles & Objectives
- Main objective: Cross levels to find the princess
- Extra lives as optional collectibles in challenging spots

### Creative Elements
- **Kiro Logo Integration**: 
  - Appears behind windows in the background
  - Flees when character approaches certain spots
  - Dynamic and animated presence

### Physics Settings (Confirmed)
- Gravity: 0.7
- Jump Power: 12
- Movement Speed: 5
- Inertia/friction for direction changes
- One screen per level
